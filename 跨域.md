#### 极简插件
- Access-Control-Allow-Origin
- 插件
![](https://images-cdn.shimo.im/ZJvWWwBT8oAIVvkf/image.png!thumbnail)

```
let url = '//api-m.mtime.cn/Showtime/HotCitiesByCinema.api';
$.ajax({
        url,
        type: 'GET',
        dataType: 'json',
        data: {},
    })
    .done((res) => {
        console.log("success", res);
    })
    .fail(() => {
        console.log("error");
    })
    .always(() => {
        console.log("complete");
    });
```
